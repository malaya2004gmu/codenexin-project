# Nyay Paksh - Login Page

A modern, responsive login page for the Nyay Paksh political party with OTP verification functionality.

# Technology Stack

- *Frontend:* React.js
- *Styling:**CSS
- *Icons:** Font Awesome
- *Routing:* React Router DOM
- *Build Tool:* Create React App
